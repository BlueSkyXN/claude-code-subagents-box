# 编程语言专家子代理

编程语言专家是特定编程语言及其生态系统的专业指导。这些子代理拥有对语言惯用法、最佳实践、性能优化技术和框架专业知识的深入理解。无论你是在使用现代 Web 框架、系统编程语言还是企业平台，这些专家都能确保你编写出地道、高效且可维护的代码。

## 何时使用编程语言专家

当你需要以下帮助时，可以使用这些子代理：
- **掌握语言特定功能**和高级模式
- **优化性能**，使用语言特定技术
- **实施框架最佳实践**，用于生产应用
- **迁移或现代化**现有代码库
- **解决语言特定挑战**，获得专家指导
- **学习高级模式**和语言惯用法
- **构建框架特定应用**，充满信心

## 可用子代理

### [**angular-architect**](angular-architect.md) - Angular 15+ 企业级模式专家

**适用场景：** 构建企业级 Angular 应用、实现复杂状态管理、优化 Angular 性能，或迁移到最新 Angular 版本。

**适用场景：** 编写高性能 C++ 代码、实现系统级软件、优化内存使用，或开发嵌入式系统。

## 入门指南

1. **确定你的技术栈**并选择合适的专家
2. **描述你的项目背景**，包括现有代码和约束条件
3. **明确你的目标**（学习、优化、实现）
4. **分享相关代码**以获得上下文感知的帮助
5. **遵循专家的指导**以获取最佳实践

## 最佳实践

- **使用语言惯用法：** 每个专家都了解编写代码的地道方式
- **利用生态系统工具：** 专家了解完整的生态系统
- **遵循框架约定：** 每个框架都有自己的最佳实践
- **尽早考虑性能：** 语言特定的优化很重要
- **考虑可维护性：** 编写未来开发者能理解的代码
